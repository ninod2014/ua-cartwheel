Detailed Description

This class implements an intermediate-level controller. Given a low level controller (of the type IKVMCController, for now), this class knows how to set a series of parameters in order to accomplish higher level behaviours.

NOTE: We will assume a fixed character morphology (i.e. joints & links), and a fixed controller structure (i.e. trajectories, components).
