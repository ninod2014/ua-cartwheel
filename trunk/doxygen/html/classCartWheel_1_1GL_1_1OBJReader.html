GLMesh * OBJReader::loadOBJFile ( const char *  fileName  )  [static]

This static method reads an obj file, whose name is sent in as a parameter, and returns a pointer to a GLMesh object that it created based on the file information. This method throws errors if the file doesn't exist, is not an obj file, etc.
