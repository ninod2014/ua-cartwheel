00027 /*=====================================================================================================================================================================*
00028  * This file implements the classes needed for the storage and display of simple 3d meshes. The classes are implemented in a way that promotes the use of vertex       *
00029  * arrays for increased efficiency.                                                                                                                                    *
00030  * At some point I might implement a better mesh data structure suitable for mesh editing (maybe DCEL (the half-Edge structre - http://www.holmes3d.net/graphics/dcel/)*
00031  * like the one used by Bruno Levy for Graphite). Until then I'll use this one...                                                                                      *
00032  * NOTE: One BIG limitation of this class is that each vertex can have only one set of texture coordinates and normal attached to it!!!! It will do for now.           *
00033  *                                                                                                                                                                     *
00034  * The following classes are implemented:                                                                                                                              *
00035  *                                                                                                                                                                     *
00036  * CLASS VertexInfo: For every vertex in every polygon, this class will keep track of the neighbours - this will be used for fast normal computations.                 *
00037  *                                                                                                                                                                     *
00038  * CLASS SharedVertexInfo: This class is used to store, for each vertex, a collection of VertexInfo classes - this way, when computing the normal at each vertex,      *
00039  *                                      we only need to iterate through this collection in order to get the normals from each polygon that contains this vertex, and then we average them  *
00040  *                                                                                                                                                                     *
00041  * CLASS GLIndexedPoly: This class contains a list of vertex indices that correspond to the vertices that make up the polygon. This class should be populated by the   *
00042  *                                      class that reads in the mesh file, and then passes it to the GLMeshObject.                                                                         *
00043  *                                                                                                                                                                     *
00044  * CLASS GLPolyIndexList: This class contains an array of tightly packed indeces of vertices that belong to polygons that have the same number of vertices. As an      *
00045  *                                      example the indices of all vertices that belong to traingles in the mesh, would be tightly packed together in one instance of this class. There    *
00046  *                                      will be one of these objects for all the triangles in the mesh, one for all the quads, and so on.                                                  *
00047  *                                                                                                                                                                     *
00048  * CLASS GLPolyCategory: The last in the set of helper classes, this one acts as a container of GLPolyIndexList, one for triangles, one for quads, and so on.          *
00049  *                                      Everytime a new polygon is added, this class is responsible with choosing the correct PolyIndexList that its vertices should be added to.          *
00050  *                                                                                                                                                                     *
00051  * CLASS GLMesh:        This is the class that has lists of vertex coordinates, texture coordinates and normals that will be used when handling the object. It implements  *
00052  *                                      methods for printing the model.                                                                                                                    *
00053  *                                                                                                                                                                     *
00054  *=====================================================================================================================================================================*/
