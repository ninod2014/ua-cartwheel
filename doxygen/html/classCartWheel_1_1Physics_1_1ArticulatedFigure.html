void ArticulatedFigure::addJoint ( Joint *  jt  ) 

Adds a joint to the figure This is an empty function as the joints are not tracked by the ArticulatedFigure. This makes it possible to disown Python of the joint pointer so that it doesn't garbage collect it. The real place where Python should be disowned is when Joint.setParent() is called since the parent is responsible for deleting the joint. However, I don't know how to force python to disown an object when a method is called.
