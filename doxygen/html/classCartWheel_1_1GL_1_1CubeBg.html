CubeBg::CubeBg ( const char *  rootFileName  ) 

This is the constructor. The file name that is passed in corresponds to the root name of the six texture files that will be loaded for this background. The file names can be obtained by adding: +x.bmp, -x.bmp, +y.bmp, etc to the root file name.
