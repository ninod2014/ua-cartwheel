Image * BMPIO::allocateImage ( BMPInfoHeader *  infoHeader,
int  nrBits = 24 
) [protected, virtual]

This method is used to allocate an image. This default method loads either an 8-bit or a 24-bit image, but it can easily be modified to load 32-bit images (RGBA) as well. The image size, nr bits, etc is loaded from the bitmap header that is passed in as a parameter.

This method is used to allocate an image for the bitmap that is to be read from a file. infoHeader - contains information regarding the size of the bitmap, etc nrBits - indicates how many bits the image will have - default value is 24, but 32 bit (RGBA) values can also be read.
